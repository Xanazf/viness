# Vite/NextUI and Express/SQLite3/Sequelize (ViNESS) Full Stack Template

## Install dependencies

```sh
npm install
# or
yarn install
```

## Run raw

```sh
npm run build && npm run start
# or
yarn build && yarn start
```

## Run with Docker

```sh
docker build -t viness .
docker run -p 3000:3000 -p 5000:5000 viness
```

## Develop

```sh
npm run dev
# or
yarn dev
```
